About this API

This API is a side project that was put online in July 2015. It is a work in progress. PROTIP: To view this API, if you haven't already, you should consider getting a plug-in that will allow you to read JSON in a readable format. If you're using Chrome, check out JSONView.

Oh my Glob!! What the stuff are you doing?

Making an API for my favorite show, Adventure Time! The data was collected from the Adventure Time Wikia Page using the Python scraper, Scrapy. Scheduling for collecting new data that comes in with each new episode is done through Celery. Everything is packaged together into this lumpy lil web app through Django and Django Rest Framework. The rest of the details can be seen on our Github page!
